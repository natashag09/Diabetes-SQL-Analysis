# Diabetes SQL Analysis Project

## Project Overview
This project performs SQL-based analysis on a diabetes prediction dataset to identify risk patterns, compare diabetic and non-diabetic patients, and generate healthcare insights.

## Tools Used
- MySQL Workbench
- SQL
- GitHub

## SQL Concepts Used
- Aggregate Functions
- GROUP BY
- CASE Statements
- Window Functions
- DENSE_RANK()
- NTILE()
- Subqueries

## Dataset Features
- Glucose
- BMI
- Blood Pressure
- Insulin
- Age
- Diabetes Outcome

## Key Insights
- Higher glucose and BMI are associated with diabetes.
- Older age groups show higher diabetic prevalence.
- High-risk patients identified through analytical filtering.
